# Blankify
Text Classification Web Application

Accepts a given text as input, and classify into one of the classification options provided.
Application exposes a REST API, which accepts array of JSON objects and respond back with corresponding classification of the input received.
Application accepts a pre-defined set of outputs, and find the closest match for any text from this list.

## Features:
- Login/Authentication for control of the application
- Model can be trained with additional inputs and outputs.
